IRC Chat System

> LATERALUS IRC — Server-Based Chat
A complete IRC-style chat system written entirely in Lateralus.
Features async networking, channels, private messages, nick management,
and a full server command protocol. Runs on the async/await runtime.

The Lateralus IRC system consists of two components: a chat server that manages connections, channels, and message routing, and a terminal client that provides the classic IRC experience. Both are written in pure .ltl.

Wire Protocol

Messages use a simple text-based protocol over TCP, inspired by IRC RFC 1459:

// Client → Server NICK spiral JOIN #lateralus PRIVMSG #lateralus :hello everyone! PRIVMSG fibonacci :nice work on codegen PART #random QUIT :goodbye! // Server → Client :server 001 spiral :Welcome to LateralusIRC :server 332 spiral #lateralus :Welcome topic :toolFan!user@host PRIVMSG #lateralus :hey! :spiral!user@host JOIN #lateralus :server 353 spiral = #lateralus :@admin spiral toolFan

Server Source (Lateralus)

Full source: examples/irc_chat.ltl

// ═══════════════════════════════════════════════ // LATERALUS IRC Server — Core Architecture // ═══════════════════════════════════════════════ import net import strings import async // IRC user connection struct IRCUser { nick: str, socket: Socket, channels: list, joined_at: int, } // Channel with users and topic struct Channel { name: str, topic: str, users: list, ops: list, } // Server state struct IRCServer { host: str, port: int, users: map, channels: map, motd: str, } // Create and start the server fn start_server(host: str, port: int) { let server = IRCServer { host: host, port: port, users: {}, channels: {}, motd: "Welcome to LateralusIRC v2.4!", } // Create default channels create_channel(server, "#lateralus", "Spiral out. Keep going.") create_channel(server, "#dev", "Development discussion") println("IRC server on ${host}:${port}") // Accept loop — spawns per-client let listener = net.listen(host, port) while true { let conn = net.accept(listener) spawn handle_client(server, conn) } }

Client Source (Lateralus)

// ═══════════════════════════════════════════════ // LATERALUS IRC Client — Terminal UI // ═══════════════════════════════════════════════ fn start_client(host: str, port: int) { println("Connecting to ${host}:${port}...") let sock = net.connect(host, port) // Read loop — receive from server spawn fn() { while true { let line = net.readline(sock) display_message(line) } } // Input loop — send user commands while true { let input = readline("> ") if starts_with(input, "/") { send_command(sock, input) } else { send_message(sock, current_channel, input) } } } fn send_command(sock, input: str) { let parts = split(input, " ") let cmd = parts[0] match cmd { "/nick" => net.send(sock, "NICK ${parts[1]}"), "/join" => net.send(sock, "JOIN ${parts[1]}"), "/msg" => net.send(sock, "PRIVMSG ${parts[1]} :${rest}"), "/quit" => { net.send(sock, "QUIT") exit(0) }, _ => println("Unknown: ${cmd}"), } }
