Pipeline Data Processing

Process a list through multiple transformations using |> .
// Count words longer than 4 chars in a sentence fn word_analysis(text: str) -> int { text |> split(" ") |> filter((w) => len(w) > 4) |> map(lower) |> distinct() |> len() }

Structs & Methods

User-defined types with fields and functions.
struct Vec2 { x: float, y: float } fn magnitude(v: Vec2) -> float { sqrt(v.x ** 2 + v.y ** 2) } fn dot(a: Vec2, b: Vec2) -> float { a.x * b.x + a.y * b.y } let v = Vec2{ x: 3.0, y: 4.0 } println(magnitude(v)) // 5.0

Enums & Traits v1.5+

Algebraic data types with trait-based polymorphism.
enum Shape { Circle(float), Rect(float, float), } trait Area { fn area(self) -> float } impl Area for Shape { fn area(self) -> float { match self { Circle(r) => 3.14159 * r ** 2, Rect(w,h) => w * h, } } }

Async/Await v1.5+

Concurrent programming with async functions and await.
async fn fetch_data(url: str) -> str { let response = await http_get(url) return response.body } async fn main() { let data = await fetch_data("https://api.example.com") println(data) }
