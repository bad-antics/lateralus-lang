Batches 92-96: systems-side sprint

Recap 3/4 — closer to the metal.

Batch 92 — Graphics

A 32bpp framebuffer with src-over compositing, BDF bitmap font loader, serpentine Floyd-Steinberg dither, Bresenham + Wu-antialiased lines and midpoint circle, plus a rotozoom demo in lateralus-examples.

Batch 93 — Protocols

CoAP per RFC 7252 with extended option-delta, MQTT with remaining-length varint and PUBLISH framing, AMQP 0-9-1 frame codec, WebSocket with mask-enforcement, and STOMP with escape/unescape.

Batch 94 — Filesystems

FAT32 BPB + cluster-chain walker, ISO 9660 PVD + directory records, POSIX ustar + GNU LongLink, cpio newc, and ext2 superblock + block-group descriptor + inode locator.

Batch 95 — OS Internals

x86-64 Linux syscall table, a Vyukov seq-lock MPMC ringbuffer, a 3-state futex mutex, POSIX signals with the self-pipe trick, and copy-on-write mmap helpers.

Batch 96 — Concurrency

Writer-preferring RwLock, timed semaphore, bounded channel, generation-counter Barrier, and a one-shot CountDownLatch.
