# LATERALUS VS Code Extension

> **This directory is deprecated.** The current VS Code extension lives at:
>
> [`/vscode-lateralus/`](../../vscode-lateralus/)
>
> Version: **1.5.0**

The `vscode-lateralus/` directory contains the full extension with:
- Syntax highlighting for `.ltl` and `.ltlml` files
- Snippets for all v1.5 language constructs
- Language configuration (brackets, comments, auto-close)
- LSP client integration
- DAP debugger integration
- File icons

## Install

```bash
# Use the workspace task:
#   "Install Lateralus VS Code Extension"
# Or manually:
cp -r vscode-lateralus ~/.vscode/extensions/lateralus.lateralus-lang-1.5.0
```
