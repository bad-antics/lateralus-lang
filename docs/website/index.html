Welcome to LATERALUS

LATERALUS is a pipeline-first programming language built for scientists, engineers, and anyone who believes code should read like thought. Named after the Tool album, it embodies the idea of spiraling outward — starting simple, growing complex, always returning to the core.

The |> pipeline operator passes data through transformations naturally. No callbacks. No method chaining. No boilerplate. Just flow.

// Hello, LATERALUS v2.4 fn main() { let result = range(1, 10) |> filter(fn(x) { x % 2 == 0 }) |> map(fn(x) { x ** 2 }) |> sum() println("Sum of even squares: ${result}") } // Enum + trait — new in v1.5+ enum Shape { Circle(float), Rect(float, float), } trait Area { fn area(self) -> float }

Key Features

Icon Feature Description
⟶ Pipeline Operator |> Chain transformations without variables. Data flows left to right.
🔷 Enum / Trait / Impl Algebraic data types, traits with methods, impl blocks. Full HM type inference.
⚡ Async / Await First-class async runtime. Spawn tasks, channels, structured concurrency.
∑ Math Engine Arbitrary precision, matrices, vectors, automatic differentiation, Newton solver.
⚗ Science Module CODATA physical constants, ODE solver (RK4), FFT, statistical distributions.
🔐 Crypto Engine SHA-256, BLAKE2b, AES-256-GCM, HMAC, LBE binary encoding.
🎯 Pattern Matching match expressions with guards, structural patterns, exhaustive checking.
🖥️ C Backend Transpile to C99. Freestanding mode for kernels and embedded. Powers LateralusOS.
📡 Event Keywords emit, probe, measure — reactive programming first-class.
💬 IRC Chat Server Built-in networking. Full IRC server + client written in pure Lateralus.

# Clone the repository $ git clone https://github.com/lateralus-lang/lateralus $ cd lateralus # Install with pip $ pip install -e . # Verify installation $ lateralus --version LATERALUS 2.4.0 # Run a program $ lateralus run hello.ltl # Start the IRC chat server $ lateralus run examples/irc_chat.ltl # Start the REPL $ lateralus repl
