How to Contribute

LATERALUS is open source and welcomes contributions of all kinds. Whether you're fixing bugs, adding stdlib functions, improving documentation, or writing examples — all contributions are valued.

AreaWhere to Start
Bug FixesCheck GitHub Issues, look for "good first issue" label
StdlibAdd functions to existing .ltl files or propose new modules
DocumentationImprove .ltlml docs, add examples, fix typos
TestsAdd to tests/ directory, improve coverage
CompilerRead CONTRIBUTING.md, understand the IR pipeline first
BootstrapHelp write v2 compiler in bootstrap/*.ltl
