Batches 82-86: five domains, 25 modules

Lateralus sprint log — recap of the first block of the capstone run.

This is the first of four recaps that close out the 100-batch sprint. It covers the transition out of the foundational stdlib (batches 1-81) and into the niche-first half of the arc.

Batch 82 — Embedded (hw/)

  • hw/adc.ltl — ring-buffered DMA ADC driver with oversampling.
  • hw/i2c.ltl — master-mode I²C bit-banger with clock stretching.
  • hw/pwm.ltl — 16-bit PWM with dead-time for half-bridge output.
  • hw/spi.ltl — four-mode SPI driver (CPOL/CPHA matrix).
  • hw/uart.ltl — 8N1 UART with software flow control.

Batch 83 — Compression

Streaming gzip, lz4 block codec, zstd frame scanner, bzip2 RLE stage, and an lzw encoder sized for GIF compatibility.

Batch 84 — Gamedev

An ecs::World, an astar planner, an aabb_tree for broadphase, a tilemap renderer, and a fixed-timestep game_loop.

Batch 85 — Finance

Black-Scholes with full Greeks, a price-time-priority order_book, a var_historical risk module, a garch_1_1 volatility estimator, and a black_litterman view-blender.

Batch 86 — Text

Unicode NFD via Hangul algorithmic decomp, Wagner-Fischer Levenshtein with Ukkonen banding, UAX #29 word/sentence segmentation, Aho-Corasick multi-pattern matcher, and a double-hashed Bloom filter with optimal m/k sizing.
